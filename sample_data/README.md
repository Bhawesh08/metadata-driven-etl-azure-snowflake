# Sample Data

This folder contains mock data examples showing the structure
of all metadata control tables used in this pipeline.

## Files

| File | Description |
|---|---|
| pipeline_config_sample.csv | Sample pipeline configurations for 5 tables |
| cdc_watermark_sample.csv | Sample watermark states showing full and incremental |
| pipeline_run_log_sample.csv | Sample run history with success and failure examples |

## Notes
- All server names replaced with placeholders
- Data is for illustration purposes only
- Use these as reference when populating your own metadata tables
